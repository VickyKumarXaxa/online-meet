# Online Meet

## Features
1. Google Sign In
2. Create New Meeting
3. Join Meeting
4. Mute/Unmute Audio
5. Mute/Unmute Video
6. Change Display Name while Joining
7. Raise Hand
12. Chat
16. Disable Camera
17. End Call
18. See Your Meeting History
19. Log out

## Tech Used
Firebase Auth, Firestore, Google Sign In, Jitsi Meet, Flutter
